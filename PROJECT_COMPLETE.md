## 🏗️ アーキテクチャ

### コンポーネント構成

```
AppController (統合管理)
├── MemberManager (メンバーCRUD)
│   └── SessionStorage API
├── TeamAssignmentEngine (チーム分けロジック)
│   └── Fisher-Yates アルゴリズム

    └── Canvas API
```
